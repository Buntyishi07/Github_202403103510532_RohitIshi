# Rohit_Ishi_PortfolioWebsite

Personal portfolio website for Rohit Ishi — Cloud Computing student at AMTICS, Uka Tarsadia University.

## How to run locally
1. Clone or download the repo.
2. Open `index.html` in your browser.

## Deploy to GitHub Pages
1. Create a new repository named `RohitIshi_PortfolioWebsite` (or `YourName_PortfolioWebsite`).
2. Add files, commit and push to `main`.
3. In GitHub: Settings → Pages → Source → Deploy from `main` branch → `/ (root)`.
4. Wait a minute — the site will be at: `https://yourusername.github.io/RohitIshi_PortfolioWebsite/`
